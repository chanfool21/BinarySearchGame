Binary Search Game


How to play?


Aim of the game is to predict a number.
The number will be in between 1 to 100
Total number of chances to predict a number will be 10 at max.
After each wromg choice, a hint message will be displayed
The message can be one of the following strings
    If you guessed right
  • The "guessed number is correct. You Won !!
    • And if you didn't guess right
  • The number is greater than what you guessed. Guess Again !!
    • OR
  • The number is less than what you guessed. Guess Again !!
If you are able to predict with in 10 chances, you win, otherwise you lose
